# design-skills

A Claude Code plugin bundling Ved's personal design system as a skill.

## Skills

- **design-agent** — applies a fixed type scale (SF Pro, 12/13/14/24px), a three-grey text hierarchy, icon sizing (14px nav / 20px card), and radii conventions (8px nav / 16px card / pill CTAs) whenever building a dashboard, UI mockup, or styled artifact.

## Install

```bash
claude plugin install rugved/design-skills
```

Or add it to a project's `.claude/settings.json` plugin list.
